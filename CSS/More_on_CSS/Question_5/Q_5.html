Google Fonts

Google Fonts is a computer font and web font service owned by Google. This includes free and open source font families, an interactive web directory for browsing the library, and APIs for using the fonts via CSS and Android. Google Fonts is also used With the Google Workspace

Montserrat is a geometric sans-serif typeface designed by Argentine graphic designer Julieta Ulanovsky and released in 2011. It was inspired by posters, signs and painted windows from the first half of the twentieth century, seen in the historic Montserrat neighbourhood of Buenos Aires.The project was started in 2010 by Ulanovsky and was released through the Google Fonts catalogue in 2011. Montserrat has become increasingly popular among web designers, and it is used on over 17 million websites.
